### 安装Selenium

使用pip安装即可

```shell
pip install selenium
```

### 下载驱动

Chrome浏览器驱动：

[ChromeDriver Mirror (taobao.org)](http://npm.taobao.org/mirrors/chromedriver/) 

[ChromeDriver - WebDriver for Chrome - Downloads (chromium.org)](https://chromedriver.chromium.org/downloads)

Edge浏览器：[Microsoft Edge Driver - Microsoft Edge Developer](https://developer.microsoft.com/en-us/microsoft-edge/tools/webdriver/)



示例：

```python
from selenium import webdriver
import time

# 指定浏览器驱动
driver = webdriver.Edge('D:\Driver\msedgedriver.exe')
# driver = webdriver.Chrome()
# driver = webdriver.Firefox()
# driver = webdriver.Safari()
# driver = webdriver.Opera()

# 打开指定url的web页面
driver.get("https://www.baidu.com")
time.sleep(5)

# 关闭当前页面
driver.close()

# 退出驱动并关闭所有页面
driver.quit()
```
